数据库语句合集之DCL

2019-07-14
知识汇总
  • 数据库
本文总阅读量次

数据库语句合集将分四大类DDL(数据定义语句)、DML（数据操纵语句）、DCL(数据控制语句)、 DQL(数据查询语句)对数据基本操作语句的进行归纳总结。本文为DCL的一些基本语句。

1、数据控制语句主要涉及grant、revoke语句

2、常用语句操作如下：

  • 授予用户权限： grant 权限 on table 表名 to 用户 with grant option

PS1: “with grant option”用于指明同意该用户把该权限授予其他用户，且不能循环授权:A->B,B->C;但C不能给A。

PS2: 若表示全部操作权限，则 权限 = all privileges， 其他情况下，权限 = select/update(列名)/delete/insert

PS3: 若表示授权给所有用户，则 用户 = public

  • 收回权限： revoke 权限 on table 表名 from 用户 cascade/restrict

PS1: cascade表示级联收回，即A->B,B->C,当A收回了B的权限后，C的权限也会被收回。默认情况为级联收回

PS2: restrict表示防止级联收回，即A->B,B->C,当A收回了B的权限后，C的权限不会被收回
