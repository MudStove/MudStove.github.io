数据库语句合集之DML、DQL

2019-07-13
知识汇总
  • 数据库
本文总阅读量次

数据库语句合集将分三大类DDL(数据定义语句)、DML（数据操纵语句）、DCL(数据控制语句)、 DQL(数据查询语句)对数据基本操作语句的进行归纳总结。本文为DML、DQL的一些基本语句。

DML: 数据操纵语言

1、数据操纵语言： 对表格中数据进行增删改的语句；包括：update、insert、delete

2、常用的操纵语句：

  • 插入数据记录： 包括插入元组（一条记录）和插入子查询结果（可插入多条元组）

    • 插入元组： insert into 表名(字段1, 字段2, … , 字段n) values(值1, 值2, … , 值n)

      PS: 若(字段1, 字段2, … , 字段n)省略，则表示插入完整的一条记录，且values值的顺序与表中顺序保持一致

    • 插入子查询结果： insert into 表名(字段1, 字段2, … , 字段n) select语句

  • 删除数据记录： delete from 表名 where 条件
  • 修改数据记录： update 表名 set 新数据 where 条件

  • 增加/插入视图中数据记录： insert into 视图名 values(值1, 值2, … , 值n)；

  • 删除视图中数据记录： delete from 视图名 where 条件

  • 修改视图中数据记录： update 视图名 set 新数据 where 条件

DQL: 数据查询语言

1、数据查询语言： 对表格中数据进行查询的语句；即 select

  • 查询数据记录：select all/distinct 目标列

    from 表
where 条件表达式
group by 列 having 条件表达式
order by 列 asc/desc

    PS: where…和group..二者取其一

  • 查询视图中数据记录： select 列 from 视图名 where 条件
