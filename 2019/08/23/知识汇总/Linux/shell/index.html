vi编辑器

1、简介

  • 使用原因

    • 有些情况下能使用的只有vi编辑器

    • 功能强大，使用很方便

    • 能很好的使用正则表达式，提高工作效率

  • 工作模式和特点

    • 可编辑无格式的文本文件

    • 是交互式的文本编辑器，没有菜单，只有命令

    • 只是文本编辑器，不能用于排版

  • 三种模式： 文本编辑模式 、 命令模式、 行编辑模式

    • 文本编辑模式–> 命令模式 by Esc键 ； 命令模式–>文本编辑模式 by A\I\S\O\C等命令

    • 行编辑模式–> 命令模式 by 执行非退出命令 ； 命令模式–>行编辑模式 by “：”键

  • vi的启动和退出

    • 启动：vi [option] 文件名/文件格式

    • 保存与退出：

      • 保存文件： :w

      • 强制保存文件： :w!

      • 保存文件后退出： :q

      • 不保存文件并强制后退出： :q!

      • 保存文件并退出： :wq

      • 打开多个文件编辑： :a

  • 命令模式下基本命令

    • 定位光标
    命令 含义
    w 光标移到下一个单词的第一个字符
    W 光标移到后面第二个单词的第一个字符
    e 或 E 光标移到下一个单词的最后一个字符
    b 或 B 光标移到前一个单词的第一个字符
    /string 光标后移到string字符串的第一个字符处
    ? string 光标前移到string字符串的第一个字符处
    $ 光标移到本行的最后一个字符处
    Ctrl+ b 光标移到本行的第一个字符处
    • 添加和插入
    命令 含义
    i 光标前插入字符
    I 当前行的行头插入字符
    a 光标后插入字符
    A 当前行的行尾插入字符
    o 当前行的行后插入新行
    O 当前行的行前插入新行
    • 删除
    命令 含义
    x 删除光标所在位置字符，并且光标移动到下一个字符
    X 删除光标所在位置的前一个字符，并且光标不移动
    D 删除光标所在位置及其之后字符，并且光标移动到行尾
    d+d 删除光标所在行，并且光标移动到下一行
    d+w 删除光标所在位置及其下一单词间字符，并且光标移动到下一单词的第一个字符处
    • 修改
    命令 含义
    s
    S
    c+c
    C 或 c $
    r
    R
    • 查找
    命令 含义
    / + 正则表达式 向后查找
    ? + 正则表达式 向前查找

  • 行编辑命令

    • 行编辑命令地址

    • 执行shell命令

    • 从文件中读取 :: r 文件名

    • 将指定内容写入文件
