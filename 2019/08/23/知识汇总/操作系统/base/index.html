操作系统

1、计算机系统 = 硬件 + 软件

  • 计算机硬件：由中央处理器、存储器、输入输出设备、控制总线、外部设备

  • 计算机软件：系统软件、应用软件

  • 计算机硬件软件和应用间是一种层次结构，由里到外：裸机、操作系统、编译软件/编辑软件 、 各种应用软件

2、操作系统类型

  • 批处理操作系统

    • 批处理：把用户提交的作业分类，同一批作业变成一个作业执行序列，每批作业由专门的监督程序专门处理

    • 特征：

      • 用户脱机使用计算机

      • 成批处理

      • 多道程序运行： 优点在于资源共享

    • 多道程序系统和多重处理系统

      • 多道程序系统：计算机内存中同时存放几道相互独立的程序，它们轮流使用CPU，轮流执行、但都处于运行状态

      • 多重处理系统：计算机内存中同时存放几道相互独立的程序，但因为有配置多个CPU，所以它们同时执行程序

  • 分时操作系统

    • 定义: 采用时间片轮流的方式、使一台计算机能为多个用户服务。但因为响应时间足够快，能给他们都提供交互会话能力。是一个联机、多用户、交互式的操作系统

    • 特征：

      • 交互性

      • 多用户同时性

      • 独立性

  • 实时操作系统

    • 定义：一直联机操作系统，需要计算机系统能够以足够快的速度处理外来信息（在允许时间范围内）

    • 特点： 即时响应、高可靠性

  • 个人计算机操作系统

  • 网络操作系统

  • 分布式操作系统

    • 定义：通过计算机网络，将物理上分布的、具有自制能力的数据处理系统或计算机系统互联起来实现资源共享和信息交换并完成任务

3、操作系统功能

  • 处理机管理

  • 存储管理

  • 设备管理

  • 信息管理（文件系统管理）

  • 用户接口
