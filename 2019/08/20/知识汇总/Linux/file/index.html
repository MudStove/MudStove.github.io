基本概念

1、文件

  • 定义：文件是存储在某一介质上的具有标识名的相关信息的集合

  • 特性：

    • 任何具有独立意义的一组信息都可以组织成一个文件

    • 可保存性

    • 可按名存取

  • 类型

    • 普通文件：文本文件、二进制文件、数据文件等

    • 目录文件：存放文件名和其他有关文件信息的文件，也是可以用于检索的文件

      • 目录文件 = n * 目录项 = n * （文件名 + 文件号/i节点号），易于检索

      PS: DOS/Windows 系统下目录也有目录项构成，但是目录项中无文件号，而是有文件属性、创建日期等，长度大于16B

    • 链接文件

      • 硬链接

      • 软链接

    • 设备文件
