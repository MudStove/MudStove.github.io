磁盘和文件常用命令介绍

1、磁盘管理

  • 查看磁盘空间使用情况： df [option]…[File]

    • -a: 所有已安装文件系统磁盘的空间使用情况

    • -T：显示文件系统类型

  • 显示磁盘上文件或目录的大小： du [option]…[File]

    • -a: 目录中所有文件的空间占用情况

    • -h：以容易理解的方式显示出文件或目录的大小

    • -S：显示该目录直接的空间占用情况，不包括子目录的情况

2、文件与目录管理

  • 目录切换：

    • 进入目录：cd 文件名

    • 退出目录：cd..

  • 显示当前目录： pwd [option]

  • 新建目录： mkdir [option] 目录列表

    • mkdir不能简写为md

    • -m:设置目录的权限 ==> mkdir -m 777 目录名

    • -p:若所要建立的目录的上级目录不存在，则一并建立

    • -v: 显示命令执行的详细过程

  • 文件或目录复制 cp [option] 需要复制的文件/目录 复制的目的地

  • 移动文件或目录 mv [option] 需要复制的文件/目录 复制的目的地

  • 删除空目录 rmdir [option]… 要删除的目录列表

  • 删除文件或目录 rm [option] 文件名/目录

    • 用rm命令删除，option一定要有-r/-R：级联删除（子目录/子文件同时删除）

    • -f: 强制删除

    • -i: 删除前需要用户确认

  • 显示目录内容 ls [option] 文件名/目录

3、查找文件

  • 命令行方式：

    • 基本用法： find 查找路径 文件名/目录名

    • 完整用法： find 查找路径 条件/表达式（逻辑表达式/命令选项/测试条件/后续动作）

  • 图形方式

4、修改文件权限

  • 命令行方式：

    • chmod [option]…模式 [模式] 文件名 eg: chmod u=rwx g=rx o=x 文件名

      • u：所有者 g: 所有组 o:其他人

    • chmod [option]…8进制模式 文件名 eg: chmod 751 文件名

    • chmod [option]…–reference = 被参考权限的文件或目录 文件名 【不常用】

  • 图形方式

5、查看文件内容

  • cat命令 cat [option]…文件名(可以多个文件)

    • text1 >> text2 : 文件追加

    • text1 > text2 : 文件覆盖

  • more命令

    • 适用于查看大型文件因为可以分屏显示

    • more [option] 文件名

6、文件压缩与解压缩

  • 命令行方式

    • compress:

      • 标准的压缩命令，压缩后原属性保持不变，扩展名为 .Z

      • -f: 强制压缩，不管需要压缩的文件是什么格式都可以被压缩为 .Z 文件

      • -v:显示压缩比率

      • -r: 递归压缩，子目录等也一起压缩

      • compress [option] [文件名或文件类型]

    • uncompress

      • 标准的解压缩命令，是指向compress程序的符号链接文件，压缩后文件属性保持不变

      • uncompress [option] [文件名或文件类型]

    • gzip

      • 压缩比率高，所有经常被使用；产生 .gz 格式文件

      • 由于gunzip为gzip的硬链接，所以gzip本身也可以解压缩

      • gzip [option] [文件名或文件类型]

    • gunzip

      • gunzip [option] [文件名或文件类型]

  • 图形方式

7、文件备份

  • tar命令 tar [option] [文件名或文件类型]

  • dump命令: 支持多级备份

8、简单的信息处理

  • wc命令

    • 显示文件列表中各文件的行数、词数等信息

    • wc [option] [文件名或文件类型]

  • grep命令

    • 查找文件中特定信息

    • grep 特定信息 [文件名或文件类型]

  • sort命令

    • 将文本文件中内容加以排序，且可以输出

    • sort [option] [文件名或文件类型]

  • pr命令

    • 修改文件中内容格式并输出

    • pr [option] [文件名或文件类型]

  • cut命令

    • 可选取并输出特定范围的文件内容

    • cut [option] [文件名或文件类型]

  • tr命令

    • 替换字符、压缩重复字符、删除字符或字符串
完
