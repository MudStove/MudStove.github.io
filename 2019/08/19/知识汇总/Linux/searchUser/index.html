查看用户登录

1、who命令

  • 查询目前登录的用户信息，可显示目前系统中所有登录的用户名单，各自的登录终端、时间、状态等信息

  • 用法： who [-w/–help/–version/……]

    • 【‘[…]’为可选项】

    • -w:显示当前是否可以联机收发信息

2、w命令

  • 用于查询当前登录用户的详细情况，包括系统启动了多久、用户正在执行的程序、现在的系统时间等

  • 用法: w [-husfV] [user]

    *【‘[…]’为可选项】

    • -h: 不显示信息头

    • -u: 不显示用户当前进程和CPU占用时间

    • -s:简洁的格式显示，不含登录时间、终端阶段作业、消耗CPU时间

    • -f: 不显示登录地点

    • -v: 显示版本信息

3、finger命令

  • 用于查找并显示用户信息，包括本地与远程的登录用户，输入的账户名称不区分大小写

  • 用法： finger [-lmsp] [user…] [user@host…]

    • -s:简洁的格式显示：只显示登录时间、真实姓名、终端名称、登录名称等

    • -l：以完整格式显示：除-s外还包括用户目录，家庭电话，邮件状态等

    • -m:禁止查找用户真实姓名

    • -p:禁止显示.plan/.project/.forward文件中信息

    • user:本地用户

    • user@host：远程用户

改变用户身份

1、su命令

  • 切换为超级用户或其他用户时暂时改变身份进行一些操作时使用

  • 从当前用户切换至其他用户（超级用户/其他用户）时需要密码，而其他用户（超级用户/其他用户）切换至当前用户则不需要

  • 用法： su [-flmp] [–help] [–version] [-] [-c command] [-s shell] [user]

    *【‘[…]’为可选项】

    • user : 切换的目标用户

2、sudo命令

  • 不切换用户，但允许用户以其他用户身份来执行命令；系统预设身份为root，其他身份可在命令或 /etc/sudoers 文件中设置

  • 该命令能让一些权限有针对性的下放，也叫做授权许可的su、较su命令来说更安全

  • 用法： sudo -V/H/L/l/K/k/v/s/S/[P/b/H/p prompt] [-c class/-] [-a auth_type] [-u userame /#uid] command
