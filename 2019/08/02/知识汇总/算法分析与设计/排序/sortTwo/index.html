简单插入排序（Insertion Sort）

1、思路描述

  • 顾名思义，‘插入’排序即为把要排序的数据插入已排好序的数据中

  • 假定第一个数为已排好序的数据

  • 待排序数据依次和已排好序的数据对比，直至找到正确位置并插入

  • 循环待排序数据，直至全部排好为止

希尔排序（Shell Sort）

1、定义

  • 又叫缩小增量法，增量(gap) = d(待排序的数 - 假定的已排序的数) 【可理解为两数间距离】，通过不断缩减距离，每组待排序数据增多，每次排好的数据量增多，从而达到排序目的。if(gap === 1)->简单插入排序

2、思路描述

  • 类似于折半 + 插入排序，循环增量 = 增量/ 2；默认：增量 = 待排序总个数 / 2

  • 每次循环，让待排序数据和（待排序数据-增量*n）的数据进行比较

  • 直到增量 = 1，进行插入排序后，得到结论
