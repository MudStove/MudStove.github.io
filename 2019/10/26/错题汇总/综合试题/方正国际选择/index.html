方正国际线上笔试题之选择

1、HTTP

  • http协议支持一定时间内的TCP连接保持，这个连接可以用于发送/接收多次请求

  • 无状态连接：协议对于事务处理没有记忆功能，即每个请求都是独立的，与前面的请求和后面的请求都是没有直接联系的。

    • 实现有状态连接：cookies, session, application

    • web = http协议 + 状态机制 + 其他机制

2、下面哪种方法创建对象时不会调用类的构造函数 ---（B、E）

      A、使用反射/Class类的newInstance方法

      B、使用反序列化、调用Java.io.ObjectInputStream的readObject方法

      C、使用new关键字

      D、使用反射/Constuctor类的newInstance方法

      E、调用对象的clone()方法

      

  • 构造函数的作用是完成对象的初始化：当程序执行到new操作符时， 首先去看new操作符后面的类型，因为知道了类型，才能知道要分配多大的内存空间。分配完内存之后，再调用构造函数，填充对象的各个域，这一步叫做对象的初始化，而选项E、B中，对象的初始化并不是通过构造函数完成的，而是读取别的内存区域中的对象的各个域来完成

  • 链接：https://www.nowcoder.com/questionTerminal/50b18353a6ad48059619d000ef6dfae5?source=relative 来源：牛客网

  • new方式创建对象，其初始化过程中可以执行带参的构造器，也可以执行无参的构造器，也就是说其初始化数值可选

  • 如果采用的是静态的newInstance（）方法，其执行的构造器一定是无参的构造器，反射执行不了带参的构造器

    • 把对象转换为字节序列的过程称为对象的序列化；序列化：实现Serializable和Externalizable接口的类、writeObject()

    • 把字节序列恢复为对象的过程称为对象的反序列化；反序列化：readObject()

3、HashMap在jdk1.8中没实现哪个接口 ---（B）

      A、Colneable        B、AbstractMap        C、Map        D、Serializable

  • HashMap 是Java程序员使用频率最高的用于映射(键值对)处理的数据类型；而AbstractMap 是抽象类，不是接口
4、synchronized关键字做同步，可以使用在什么上 ---（D）

      A、对象           B、原子类型           C、方法和对象           D、方法

  • synchronized有两种用法：定义方法 和 synchronized块
5、父类子类调用顺序

  • 调用顺序取决于虚拟机编译的先后顺序，静态的东西一般都是先实例化的，然后接下来是构造方法和普通方法；所以调用顺序为：

    • 父类静态方法 –> 子类静态方法 –> 父类成员方法 –> 父类构造方法 –> 子类成员方法 –> 子类构造方法
6、下面说法错误的是 ---（B）

      A、进程是系统进行资源分配和调度的基本单位，而线程是CPU调度和分配的基本单位

      B、线程也拥有自己的系统资源

      C、一个进程中的多个线程共享资源

      D、一个线程可以创建和撤销另一个线程

  • 线程是进程中的一个实体，是被系统独立调度和分派的基本单位，线程自己不拥有系统资源，只拥有一点儿在运行中必不可少的资源，但它可与同属一个进程的其它线程共享进程所拥有的全部资源；所以线程只是占用那么一点点系统建立线程的系统资源，真正占用系统资源的是程序的执行/cpu
7、上网时发现网页不能访问，QQ使用正常，则原因可能是 ---（A）

      A、DNS问题           B、IP地址冲突           C、网线问题           D、网关冲突

  • QQ: 直接通过IP访问 ; 网页：需要域名解析成IP地址才能访问
8、此命令的运行时间是：02 08 * * 0/user/local/founder/start.sh ---（A）

       A、每周日08：02分        B、每周一02：08分        C、每周一00：08分        D、每周二08：00分

  • 命令组成： 分钟（0-59） 小時（0-23） 日期（1-31） 月份（1-12） 星期（0-6） 运行文件

    • 0 代表星期天 ，前五项中若无定义，则用 * 代替
9、Java/C/C++中volative关键字满足哪几个特性 ---（A、D）

      A、可见性           B、一致性类型           C、原子性           D、有序性

  • 解释可移步此处
10、有关于线程的叙述哪些是对的 ---（C、D）

      A、当一个线程因为抢先机制而停止运行，它被放在可运行队列的前面

      B、一旦一个线程被创建，它就立刻开始运行

      C、一个线程可能因为不同的原因停止并进入就绪状态

      D、使用start()方法可以使一个线程成为可运行的，但是它不一定立即开始运行

11、以下关于集合说法正确的是 ---（A、B、D）

      A、Map中，若key相同，则先前的值会被后放入的值替换掉

      B、Set集合中的元素是无序的，所以不能使用普通的for循环进行遍历，可以使用迭代器和foreach进行遍历

      C、Set中可以存放重复元素

      D、容器类仅能持有对象引用(指向对象的指针),而不是将对象信息 copy 一份至数列某位置。一旦将对象置入容器内,便损失了该对象的型别信息

12、Java中的集合类包括ArrayList、LinkedList、HashMap等类，下列关于集合类的说法正确的是 ---（A、B、C）

      A、ArrayList和LinkedList均实现了Lisk接口

      B、HashMap实现了Map接口，它允许任何类型的键和值对象，并允许将null用作键或值

      C、ArrayList的访问速度比LinkedList快

      D、添加和删除元素时，ArrayList的表现更佳

  • ArrayList采取数组实现，添加和删除元素时慢，查询快
