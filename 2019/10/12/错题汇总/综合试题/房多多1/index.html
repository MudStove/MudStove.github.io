选择

1、在计算机系统中以下哪些是线程和进程共用的 (可多选) ---（）

      A、地址空间        B、堆        C、栈        D、代码段

2、对于一台计算机中的一块存储了 1TB 数据的普通机械硬盘(SATAII 7.2K rpm），现在有程序随机访向其中的任何 1KB 的数据，那么每秒钟可能的平均访问次数，下列数字中最可能接近的是 ---（）

      A、10次        B、100次        C、1000次        D、10000 次

3、在一个单CPU、内存足的计算机上，对 1GB 的4字节整数数组进行quicksort排序，大约需要多长时间? ---（）

      A、10秒        B、30秒        C、60秒        D、100秒

4、下列 Linux 常用命令工具中哪些对服务器性能检查有用(可多选) ---（）

      A、iostat        B、vmstat        C、netstat        D、top

5、在客户端(Client)和服务端(Server)的跨网络服务接口调用中(即:远程过程调用RPC）以下描述符错误的是 ---（B、E）

      A、客户端在调用前会主动发起与和服务端建立连接，如TCP连接

      B、客户端会将调用的接口标识和其参数打包编码后发送给服务端，编码格式可以由客户端和服务端灵活约定

      C、服务端侧程序常常采用多线程实现对并发的支持

      D、调用的返回结果由客户端侧产生

6、When you create Java or C++ object by using "new" , and have a reference to point to the object.Please answer: (1) Where is the place the object itself lives in? (2)Where is the reference? ---（）

      A、The Stack,The Heap        B、The Stack,Register        C、The Heap,Register        D、The Heap,The Stack

7、我们来玩一个猜数字游戏:我随意写下一个位[0，1000000000]间的数字但不告诉你，现在你来猜我写的是什么数字。你每次猜测一个等案后， 我会告诉你小了、大了或对了。你在多少次之内一定能猜对 ---（）

      A、30次        B、 256次        C、 1000000次        D、 1000000000次

8、以下哪些语句可以判断整数 n 为2的幂?(2的幂:2^2、2^3、...... 2^m, 可多选) ---（）

      A、n&(n-1) == 0        B、n == (n&-n)        C、n/2 == n%2        D、(n/2)%2 == 0

9、以下哪些操作的时间复杂度是 O(n*log n)?(可多选) ---（）

      A、堆排序        B、最糟情况下的快速排序(quick sort)        C、在二叉树中查找一个数        D、归并排序(或合并排序 merge sort)

10、下对代码执行的时间复杂度是 ---（）

      A、O(xyz)        B、O(2^x2^y2^z)        C、O(2^x)        D、O(x!)

11、在一种RDBMS系统中(以MySQL为例，并采用了InnoDB作为存储引擎)，我们建立了一张表t1 (其Schema包含4个列cl,c2, c3, c4)。其Primary Key指定为列cl,并在该表的列c2上建立了B+Tree索引。我们向该表中写入了3000万条记录。问下列SQL查询语句中，哪条SQL语句理论上执行效率最高? ---（B、E）

      A、 select * from t1 where c2=”abc” and c3>1000 and c3<1050;

      B、 select * from t1 where c3>1000 and c3<1005;

      C、select * from t1 where c1=”abc” and c3>1000 and c3<1050;

      D、 select * from t1 where c1>”abc” and c1<”abz” and c2>”a” and c2< “bbb”;

12、类P是一个抽象类，类F是一个继承P的非抽象类，以下程序语句正确的是(可多选) ---（）

      A、P p = new F();        B、P p = new P();        C、F f = new F();        D、F f = new P();

13、下列C/C++代码片段中哪段是可以正确执行的 ---（）

      A、

void GetMemory(char *p) {
    p = (char *)malloc(100);
}
void Test(void) {
    char *str = NULL;
    GetMemory(str);
    strcpy(str,"hello world");
    printf("%s", str);
}

       B、

void *GetMemory(void) {
    char p[] =  "hello world";
    return p;
}
void Test(void) {
    char *str = NULL;
    str = GetMemory();
    printf("%s", str);
}

       C、

void GetMemory(char **p, int num) {
    *p = (char *)malloc(num);
}
void Test(void) {
    char *str = NULL;
    GetMemory(&str, 100);
    strcpy(str,"hello");
    printf("%s", str);
    free(str);
}
14、给你几万条用户数据，每条用户数据的电话号码是其唯一的关键字段，请选择合适的数据结构， 将这些用户数据加载到内存中管理起来。 这个数据结构支持基于电话号码的快速查询，同时又支持新数据的快速插入(有新的用户不断注册进来）：(可多选) ---（）

      A、链表        B、Hash表        C、 B+ Tree        D、队列

15、在房多多，有很多工作人员检查二手房源是不是属于虚假房源，假设所有二手房源的真实率到达了98%, 工作人员将一个真实二手房源检验为虚假二手房源的概率为2%.而一个虚假二手房源被检验为真实二手房源的概率为5%。 那么一个被检验为真实的二手房源确实是真实的概率是多大? ---（）

      A、0.9991        B、0.9989        C、0.9855        D、0.96
