选择

1、下面特性中，链表不具有的特点有 ---（）

      A、不必事先估计存储空间

      B、可随机访问任一元素

      C、插入、删除操作不需要移动元素

      D、所需空间与线性表的长度成正比

2、设计中划分模块的一个准则是 ---（）

      A、低内聚低耦合           B、低内聚高耦合           C、高内聚低耦合           D、高内聚高耦合

3、下列协议中，属于应用层的协议有 ---（）

      A、TCP           B、DNS           C、POP3           D、Telnet           E、UDP           F、ICMP

4、关于数字签名，错误的描述是 ---（）

      A、数字签名可以保证数据的完整性

      B、发送方无法否认自已签发的消息

      C、接受方可以得到发送方的私钥

      D、接受方可以确认发送方的身份

5、指针p的定义如下： char *p = (char *) malloc(100),在32位系统中，sizeof(p)大小是 ---（）

      A、1           B、4           C、8           D、100

6、Unix系统中，哪些可以用于进程间的通信 ---（）

      A、Socket           B、共享内存           C、消息队列           D、信号量

7、静态变量通常存储在进程哪个区 ---（）

      A、栈区           B、堆区           C、全局区           D、代码区

8、浏览器访问某页面，HTTP协议返回状态码403时表示 ---（）

      A、找不到该页面           B、禁止访问           C、内部服务器访问           D、服务器繁忙

9、当多个事务并发执行时，数据库系统应该保证一个事务的执行结果不受其他事务的干扰。这一特性被称为事务的 ---（）

      A、原子性           B、一致性           C、持久性           D、隔离性

10、对于派生类的构造函数，在定义对象时构造函数的执行顺序为? ---（）

      A、123           B、231           C、321           D、213

11、以下代码的输出结果为 ---（）

      A、Load B1 Load B2 Create B Create A Load A

      B、Load B1 Load A Load B2 Create B Create A

      C、Load B2 Load B1 Create A Load A Create B

      D、Load A Load A Load B2 Load Bl Create B

12、不能用来修饰interface的有 ---（）

      A、private           B、public           C、protected           D、static
