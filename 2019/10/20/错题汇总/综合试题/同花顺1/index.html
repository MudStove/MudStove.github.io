错题汇总
  • 综合试题
本文总阅读量次
1、移动开发或PC开发中，遇到哪些兼容性问题，都是怎么解决的，请举三个案例

  • js声明变量时，const可以在谷歌上使用，却不能在IE上 ==> 使用 var 替代

  • （双边距问题）一个块级元素同时使用float和margin定义后，在IE6显示下margin比设置的大 ==> 使用display:inline将其转化为行内属性

  • 高度设置较小的值时（<10px）,IE6、IE7下高度超出自己设置的高度 ==> 给超出高度的标签设置overflow:hidden;或者设置行高line-height小于你设置的高度

  • 图片间默认有间距 ==> img使用float布局

  • 不同浏览器间，margin、padding的取值不同 ==> margin: 0; padding: 0;

2、有两个实现相同功能的函数A、B，请问如何去检验两个函数的执行效率？
3、有如下变量： var foo = {'a':1},如何实现foo.fn('a')为获取foo属性为a的值，foo.fn('a':2)为设置foo的a属性的值为2？如果要在所有对象上添加fn函数功能，该如何实现？
4、正则匹配，以a开头，中间任意个b，以c结尾的字符串
5、有变量var date = new Date() ; 如何实现date.nextDay()可返回date的第二天日期
6、如何在页面中动态加载一个scipt.js文件，并在加载完毕后执行回调函数fn
7、有如下两段代码，请问执行完js代码后，会触发浏览器的重绘还是回流？请说明理由
8、有如下几个变量，请分别说明这5个变量存储在内存地址中的堆上还是栈上，并说明理由
9、什么是边框塌陷？你是怎么解决的？
