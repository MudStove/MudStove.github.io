      A、parseFloat方法：该方法将一个字符串转换成对应的小数

      A、JavaScript是一种动态类型，弱类型，基于原型，直译式的脚本语言

17、用于播放HTML5视频文件的正确HTML5元素是: ---（）

      A、< movie >           B、< media >           C、< video >           D、< canvas >

18、新窗口打开网页，用到以下哪个值 ---（）

      A、_self           B、_blank           C、_top           D、_parent

19、 以下选项中哪些属于原生js操作数组的方法 ---（）

      A、splice           B、shift           C、push           D、sort

20、以下能检测一个变量是一个String类型的是 ---（）

      A、typeof(obj) === “string”

      C、obj.constructor === String

      D、obj is String

21、创建数组方法有 ---（）

      A、let array = new []

      B、let array= new Array()

      D、let array = {}

      B、0?1:2
