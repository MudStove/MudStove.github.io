1、default、public、protected、private的作用范围大小：

  • default: 表示默认，只有同一个包内的类可以访问

  • public: 所有包都可以访问

  • protected: 表示同一个包和其子类可以访问

  • private: 只有自己的类可以访问，子类能继承，但不能调用 ==> 由小到大： private、default、protected、public

6、http的缓存机制:

https://www.cnblogs.com/DiDiao-Liang/articles/8308995.html

  • 缓存：浏览器缓存、数据库缓存、服务器缓存

  • 强制缓存: 浏览器可以直接缓存服务器返回的数据，下次使用时只要数据还没失效就都可以使用

    • 判断是否失效：Expires【服务器端返回的到期时间】 + Cache-Control 【可缓存的时间长度】

  • 对比缓存：浏览器缓存服务器返回的数据和缓存标识，下次使用时通过对比缓存标识来判断是否数据可以使用

    • Last-Modified：第一次请求时，服务器返回资源的最后修改时间

    • If-Modified-Since：再次请求时，客户端告诉服务器上次请求时返回的资源的最后修改时间

    • Etag：第一次请求时，服务器返回的唯一标识

    • If-None-Match：再次请求时，客户端告诉服务器上次请求时返回的唯一标识

7、关于cookie安全机制，哪些设置可以提高安全性？

  • 对保存到cookie里面的敏感信息加密

  • 设置HttpOnly为true

  • 设置Secure为true,保证cookie在https层面传输

  • 给Cookie设置有效期

  • 给Cookies加个时间戳和IP戳，实际就是让Cookies在同个IP下多少时间内失效

8、http协议

  • 请求报文 = 状态行 + 请求头 + 消息主体

    • 状态行 = 请求方法 + URL + 版本信息

    • 请求头：http://www.cnblogs.com/Hellorxh/p/10867892.html

      请求头 说明 示例
      Accept 可接受响应数据类型 Accept: application/json, text/plain, /
      Accept-Encoding 可接受的响应内容的编码方式 Accept-Encoding: gzip, deflate, br
      Accept-Language 可接受的响应内容语言列表 Accept-Language: zh-CN,zh;q=0.9
      Authorization 表示HTTP协议中需要认证资源的认证信息 Authorization: Basic ……
      Connection 客户端（浏览器）想要优先使用的连接类型 Connection: keep-alive
      Cookie 一个HTTP协议Cookie Cookie: localeLanguage=zh-CN; accessTo……
      Content-Type 请求体的MIME类型 （用于POST和PUT请求中） Content-Type: application/x-www-form-urlencoded
      Host 表示服务器的域名以及服务器所监听的端口号 Host: localhost:8090
      Refer 表示浏览器所访问的前一个页面，可以认为是之前访问页面的链接将浏览器带到了当前页面。 Referer: http://localhost:8090/
      User-Agent 浏览器的身份标识字符串 User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64)……

  • 响应报文 = 状态行 + 响应头 + 响应正文

    • 状态行：版本Version + 状态码 + 回应短语

    • 响应头：https://www.cnblogs.com/10000miles/p/9220537.html

      响应头 说明 示例
      Accept-Ranges 表明服务器支持哪种类型的分段请求 Accept-Ranges: bytes
      connection 连接状态？ connection: close
      content-length 响应体的长度 content-length: 5876
      content-type 返回内容的MIME类型 content-type: application/json;charset=UTF-8
      date 原始服务器消息发出的时间 date: Sun, 20 Oct 2019 09:48:03 GMT
      ETag 请求变量的实体标签的当前值 ETag: “737060cd8c284d8af7ad3082f209582d”
      Expires 响应过期的日期和时间 Expires: Thu, 01 Dec 2010 16:00:00 GMT
      Last-Modified 请求资源的最后修改时间 Last-Modified: Tue, 15 Nov 2010 12:45:26 GMT
      set-cookie 设置Http Cookie set-cookie: rememberMe=deleteMe; Path=/; Max-Age=0; Expires=Sat, 19-Oct-2019 09:48:04 GMT
      Transfer-Encoding 文件传输编码 Transfer-Encoding:chunkedess
9、document的方法

  • Document 对象是 Window 对象的一部分，可通过 window.document 属性对其进行访问

    属性或方法 作用
    cookie 返回与当前文档有关的所有 cookie
    domain 返回当前文档的域名
    lastModified 返回当前文档的最后修改的日期和时间
    referrer 返回载入当前文档的文档的 URL
    title 返回当前文档的标题
    URL 返回当前文档的 URL
    close() 关闭用 document.open() 方法打开的输出流，并显示选定的数据
    getElementById() 返回对拥有指定 id 的第一个对象的引用
    getElementsByName() 返回带有指定名称的对象集合
    getElementsByTagName() 返回带有指定标签名的对象集合
    open() 打开一个流，以收集来自任何 document.write() 或 document.writeln() 方法的输出
    write() 向文档写 HTML 表达式 或 JavaScript 代码。
    writeln() 等同于 write() 方法，不同的是在每个表达式之后写一个换行符。
